# gbit-ecosystem

Landing page showcasing the GBIT CLI ecosystem — gbit-next, gbit-start, gbit-readme, gbit-address and create-gbit-app.

## 🚀 Instalação

```bash
npm install
```

## ▶️ Como rodar

```bash
npm run dev
```

## 📜 Scripts disponíveis

| Comando | Descrição |
|---|---|
| `npm run dev` | Inicia o projeto em modo de desenvolvimento |
| `npm run start` | Inicia o projeto em modo de produção |
