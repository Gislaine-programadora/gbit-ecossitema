Web3 · Testing

gbit-address

Generate Ethereum test wallets — address, private key and mock balance — instantly, for Hardhat, Ganache or Foundry.

npx gbit-address
Documentation

gbit-readme

Scans your package.json and folder structure, then writes a complete, professional README.md — badges included.

npx gbit-readme
Any repository

gbit-start

Clone a repo, install dependencies (resolving version conflicts automatically), and get a running local URL.

npx gbit-start <repo-url>
Next.js

gbit-next

Scaffolds a Next.js + TypeScript + Tailwind app with ready-made templates — Dashboard, Ecommerce, SaaS, CRM, ERP.

npx gbit-next
Full stack

create-gbit-app

Spins up a complete project — backend, frontend (Vite + TypeScript) and smart contracts — in one command.

npx create-gbit-app my-app
